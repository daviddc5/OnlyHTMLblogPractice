Website Design System

A guide to the visual style, components, and typography for this website.

Colors

Primary Green

#344a40

Used for main headers and backgrounds.

Accent Blue

#007bff

Used for links, headings, and table headers to draw attention.

Light Background

#f8f9fa

Provides a clean and readable background for the main content.

Primary Text

#212529

The main color for body text to ensure high readability.

Text Styles

H1: Main Heading

  • Font-family: 'Roboto', sans-serif
  • Font-size: 2.5rem
  • Font-weight: 700 (bold)
  • Color: #007bff

H2: Subheading

  • Font-family: 'Source Sans Pro', sans-serif
  • Font-size: 2em
  • Font-weight: 700 (bold)
  • Color: #007bff

P: Paragraph Text

  • Font-family: 'Roboto', sans-serif
  • Font-size: 1rem
  • Line-height: 1.6
  • Color: #212529
