HTML Cheat Sheet

A quick reference guide for HTML elements and attributes.

HTML Basics & Document Structure

These tags form the basic skeleton of any HTML page.

Tag Description
<!DOCTYPE html> Defines the document type and version of HTML.
<html> Root element of an HTML page.
<head> Contains meta-information about the document.
<title> Sets the title of the document, shown in the browser tab.
<meta> Defines the meta information of a website
<title> Specifies the title for the browser tab.
<body> Contains all the content visible to the user.

Text Formatting

Use these tags to format text within your HTML document.

Tag Description
<h1> to <h6> Defines headings, with <h1> being the highest level.
<p> Defines a paragraph.
<br> Inserts a line break.
<strong> Defines important text (bold).
<em> Defines emphasized text (italic).

Links and Images

Use these tags to create hyperlinks and embed images.

Tag Description
<a href="URL"> Creates a hyperlink to the specified URL.
<img src="URL" alt="description"> Embeds an image with a source URL and alternative text.

Tables

Use these tags to create tables in your HTML document.

Table Example

Tag Description
<table> Defines a table.
<tr> Defines a table row.
<th> Defines a header cell in a table.
<td> Defines a standard cell in a table.
